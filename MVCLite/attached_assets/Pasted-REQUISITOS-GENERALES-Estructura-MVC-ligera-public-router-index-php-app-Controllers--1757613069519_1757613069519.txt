MODELO DE DATOS (mapear desde mi Excel)
Tablas y campos (ignora columnas “Columna1/2/3” del Excel):

1) campaigns
  - id INTEGER PRIMARY KEY AUTOINCREMENT
  - medio TEXT                -- (Excel: MEDIO)
  - programa_interes TEXT     -- (PROGRAMA DE INTERES)
  - tipo_campana TEXT         -- (Tipo de campaña)
  - nomenclatura_programa TEXT
  - modalidad_estudio TEXT    -- (Modalidad de Estudio)
  - linea TEXT                -- (Linea)
  - facultad TEXT             -- (Facultad)
  - nomenclatura_modalidad TEXT
  - nomenclatura_linea TEXT
  - nomenclatura_tipo_campana TEXT
  - nomenclatura_modalidad3 TEXT
  - nombre_sf TEXT            -- (Nombre SF)
  - url_programa TEXT         -- (URL programa)
  - lista_pardot TEXT         -- (Lista Pardot)
  - nombre_campania TEXT      -- (Nombre de Campaña)
  - created_at TEXT, updated_at TEXT

2) ad_groups
  - id INTEGER PRIMARY KEY AUTOINCREMENT
  - campaign_id INTEGER NOT NULL REFERENCES campaigns(id) ON DELETE CASCADE
  - fsa TEXT                  -- (FSA)
  - cohorte TEXT              -- (COHORTE)
  - tipo_publico TEXT         -- (Tipo de Público)
  - numero_grupo INTEGER      -- (Numero de Grupo de Anuncio)
  - nomenclatura_pardot TEXT  -- (Nomenclatura Pardot)
  - nombre_grupo TEXT         -- (Nombre de Grupo de Anuncio)
  - created_at TEXT, updated_at TEXT
  INDEXES: (campaign_id), (numero_grupo)

3) ads
  - id INTEGER PRIMARY KEY AUTOINCREMENT
  - ad_group_id INTEGER NOT NULL REFERENCES ad_groups(id) ON DELETE CASCADE
  - tipo_anuncio TEXT         -- (Tipo de Anuncio)
  - numero_grupo INTEGER      -- (Numero de Grupo de Anuncio)
  - nomenclatura_pardot TEXT  -- (Nomenclatura Pardot)
  - nombre_anuncio TEXT       -- (Nombre de Anuncio)
  - created_at TEXT, updated_at TEXT
  INDEXES: (ad_group_id), (numero_grupo)

FORMULARIOS (CRUD)
- Campañas: crear/editar con todos los campos de la tabla campaigns. Validar requeridos: medio, programa_interes, tipo_campana, nombre_campania.
- Grupos: dropdown para seleccionar campaña (campaign_id); resto de campos como arriba.
- Anuncios: dropdown para seleccionar grupo (ad_group_id); resto de campos como arriba.
- Botones de Editar y Eliminar (confirmación). Borrado real (DELETE) está bien.
